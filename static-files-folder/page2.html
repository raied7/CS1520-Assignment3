Credit Riser

In May 2021, I attended the Software Engineering Summit for Capital One at the end of which I participated in a 24-hour hackathon. We were assigned with the task of creating something to improve the banking experience, specifically for those without a bank. My team decided to create a web app that would help improve the user's credit score by suggesting actions to take depending on the user's input. Our web app relied on machine learning to recognize patterns from data of users who were able to improve their credit score by taking certain actions. We also employed sentimental analysis on the user input to suggest a personalized output. We ended up getting 2nd place.

The tools we used for this project:
  1. Python
  2. HTML
  3. CSS
  4. Flask
  5. Bootstrap
  6. Github
  7. Heroku
My personal takeaways from the project:
  • Led the front-end development of the website
  • Gained experience using Flask and Bootstrap frameworks
  • Practiced using Python within the machine learning/sentimental analysis context
